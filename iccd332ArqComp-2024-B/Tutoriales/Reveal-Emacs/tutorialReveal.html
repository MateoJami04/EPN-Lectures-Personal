Tutorial: Presentaciones Reveal.js desde Emacs

Lenin G. Falconí

Periodo 2026-A

Introducción

¿Qué es Reveal.js?

  • Es un framework HTML para crear presentaciones interactivas.
  • Se ejecuta en el navegador: no requiere LaTeX ni compilación a PDF.
  • Desde Emacs + Org-mode, se exporta con un solo comando.
  • El paquete ox-reveal traduce tu archivo .org a .html listo para mostrar.

Requisitos

  • El paquete ox-reveal instalado en Emacs (ya incluido en el init.el del curso).
  • Conexión a internet (Reveal.js se carga desde CDN — sin instalar nada localmente).
  • Nada más: ni LaTeX, ni Pandoc, ni compiladores extra.

El Encabezado del Archivo

Encabezado mínimo

Este es el bloque que debe ir al inicio de todo archivo .org para reveal: 

#+OPTIONS: num:nil toc:nil timestamp:nil
#+REVEAL_ROOT: https://cdn.jsdelivr.net/npm/reveal.js
#+REVEAL_THEME: dracula
#+REVEAL_TRANS: slide
#+REVEAL_INIT_OPTIONS: width:1200, height:900, margin: 0.1

#+TITLE: Mi Presentación
#+AUTHOR: Tu Nombre
#+DATE: 2026
#+LANGUAGE: es

Opciones de encabezado

Opción Descripción Ejemplo
REVEAL_ROOT URL o ruta local de reveal.js CDN o ./reveal.js
REVEAL_THEME Tema visual black, white, league, sky, beige, simple, serif, blood, night, moon, solarized, dracula
REVEAL_TRANS Transición entre slides slide, convex, concave, zoom, none
REVEAL_PLUGINS Plugins adicionales (reemplaza defaults: markdown, zoom, notes) (math) — pero MathJax ya viene por defecto, no hace falta
REVEAL_INIT_OPTIONS Configuración de reveal tamaño, márgenes, escala

Matemáticas con MathJax

MathJax se carga automáticamente — no necesitás agregar nada al encabezado. Las ecuaciones se escriben con notación LaTeX estándar de Org-mode:

  • Inline: \(E = mc^2\) produce \(E = mc^2\)
  • Display: \[ ... \] centra la ecuación en su propia línea
  • Más ejemplos en la sección de matemáticas.

Estructura de Diapositivas

Slides y secciones

  • Un * (nivel 1) crea una sección (slide horizontal).
  • Un ** (nivel 2) crea una diapositiva dentro de la sección.
  • Las secciones generan títulos visibles en la presentación.
  • Navegación: flechas ← → entre slides, ↑ ↓ entre secciones.
 * Sección 1
 ** Diapositiva 1
 Contenido aquí...

 ** Diapositiva 2
 Más contenido...
 * Sección 2
 ** Diapositiva 3

Listas y viñetas

Simplemente usá la sintaxis normal de Org-mode: 

 ** Título de la diapositiva
 - Elemento 1
 - Elemento 2
   - Sub-elemento anidado
 - Elemento 3

Fragmentos: Revelar por Partes

Fragmentos

  • Usá #+ATTR_REVEAL: :frag (appear) al inicio de una diapositiva.
  • Cada ítem de la lista aparece al presionar →.
  • Este mismo slide usa fragmentos: ¿viste cómo aparecen uno por uno?
  • Muy útil para explicar paso a paso.

Tipos de fragmentos

  • appear — aparece súbitamente
  • fade-in — aparece con desvanecimiento (este es fade-in)
  • fade-out — se desvanece
  • highlight-red — resalta en rojo
  • grow — crece desde el centro

Imágenes y Columnas

Insertar una imagen

Para una imagen centrada con ancho definido: 

#+ATTR_HTML: :width 60% :align center
[[./figuras/mi-imagen.png]]
  • :width controla el tamaño (porcentaje o píxeles).
  • :align controla la posición (center, left, right).
  • La imagen puede ser local o URL remota.

Texto a la izquierda, imagen a la derecha

Para dos columnas usamos HTML con flexbox (más fiable que float en reveal.js): 

#+begin_export html
<div style="display:flex; align-items:center; gap:30px;">
<div style="flex:1;">
#+end_export
- Acá va el texto.
- Explicación de la imagen.
- Notas adicionales.
#+begin_export html
</div>
<div style="flex:0 0 45%;">
#+end_export
[[./figuras/emacsJoke.jpg]]
#+begin_export html
</div>
</div>
#+end_export

Ejemplo real: Emacs

  • Emacs: el editor que lo hace todo.
  • Imagen a la izquierda, texto a la derecha.
  • Acá usamos flexbox, no float.

Bloques de código

Los bloques #+begin_src de Org-mode se exportan con resaltado de sintaxis:

Exportar a HTML

Cómo exportar

Para generar el archivo .html desde Emacs:

  1. Abrí tu archivo .org.
  2. Ejecutá: C-c C-e R B
    • C-c C-e → menú de exportación.
    • R → Reveal.js (R mayúscula).
    • B → exportar a archivo y abrir en navegador.
  3. Alternativa por comando: M-x org-reveal-export-to-html-and-browse
  4. El .html se genera en la misma carpeta.

Si el navegador no abre

  • En WSL (Windows): verificá que wslview esté instalado (sudo apt install wslu).

  • Alternativa WSL: usá explorer.exe como browser: 

      #+begin_src emacs-lisp
  (setq browse-url-generic-program "explorer.exe")
  (setq browse-url-browser-function 'browse-url-generic)
  #+end_src
  • En Linux nativo: xdg-open viene por defecto y debería funcionar.
  • Si nada funciona, abrí el .html manualmente desde el explorador de archivos.

Atajos de exportación

Atajo Comando equivalente Acción
C-c C-e R B org-reveal-export-to-html-and-browse Exportar y abrir en navegador
C-c C-e R R org-reveal-export-to-html Exportar a .html
